欢迎使用 Macromedia Flash 8

欢迎使用 Macromedia® Flash®，这是业界创作丰富互动内容的首选工具。Flash Professional 8 和 Flash Basic 8 为网页设计人员和开发人员提供了一系列工具，包括图形效果工具、动画制作工具、文本、视频和音频工具，用于提供从桌面到手机的丰富的用户体验。使用 Flash 创建功能强大的互动式营销、演示文稿、丰富的媒体广告、电子教学、娱乐、商业应用、互动视频体验等内容。   

全世界有超过二百万专业创作人员在使用 Flash，网络上的用户超过五亿六千万，是世界上使用最广泛的软件平台。Macromedia Flash Player 8 是表现力强且广为应用的运行时环境，适用于所有主要平台和 Web 浏览器上的外观和运行性能都一致的互动式内容和应用程序。Flash Player 8 以轻负荷、安全的运行时环境提供了无与伦比的性能；在 9 年多的时间内，它的 8 个主要版本创造了无瑕疵的安全历史记录。Flash Player 将继续与各主要合作伙伴（包括 Microsoft®、Apple®、Netscape® 和 AOL®）一起经销，从而为全球 98% 可访问 Internet 的桌面用户立即提供丰富多彩的内容和应用程序。

我们提供了一些重要的资源，可帮助您找到有关 Flash Basic 8 和 Flash Professional 8 的更多信息、获得支持以及向工作组发送反馈。请注意，尽管本自述文件包含了与这两种产品相关的信息，但您只会收到已购买了其许可证的产品。
